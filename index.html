Wiimmfi Watcher

An Android app that displays Mario Kart Wii room data by entering a friend code.

Originally a school project, I saw a use for it in the real world since the Wiimmfi website doesn't have a mobile friendly view. It is available on the Google Play Store with it's source code available on GitHub for Android devices running KitKat 4.4 or newer.
